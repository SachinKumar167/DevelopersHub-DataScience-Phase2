# Task 3 - Energy Consumption Data Analysis

## Objective
Analyze the Household Power Consumption dataset using Python and perform basic Exploratory Data Analysis (EDA) with visualizations.

## Dataset
- **Dataset Name:** Household Power Consumption
- **Source:** Kaggle
- **Records:** 2,075,259
- **Features:** 9

## Tools & Libraries
- Python
- Pandas
- Matplotlib

## Tasks Performed
- Loaded the dataset
- Explored dataset structure
- Displayed first five rows
- Checked data types
- Identified missing values
- Generated statistical summary
- Converted date and time into datetime format
- Converted power values to numeric format
- Removed missing values for analysis
- Created visualizations

## Visualizations
- Line Plot of Global Active Power over Time
- Histogram of Global Active Power Distribution
- Scatter Plot of Global Active Power
