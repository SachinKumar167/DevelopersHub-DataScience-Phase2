# Task 2 - Mall Customer Data Visualization

## Objective
Perform Exploratory Data Analysis (EDA) on the Mall Customers dataset and create visualizations to identify customer demographics and spending behavior.

## Dataset
- **Dataset Name:** Mall Customers Dataset
- **Records:** 200
- **Features:** 5

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## Tasks Performed
- Loaded the dataset
- Explored dataset structure
- Displayed the first five records
- Checked data types and missing values
- Generated descriptive statistics
- Visualized customer age distribution
- Analyzed annual income versus spending score
- Compared customer distribution by gender

## Visualizations
- Histogram of Age Distribution
- Scatter Plot: Annual Income vs Spending Score
- Count Plot of Gender Distribution
